# GenAI Video Startups Data Analysis

## Project Goals
This project aims to analyze data related to video startups leveraging GenAI technologies. Our goals include understanding market trends, identifying key players, and extracting valuable insights from data.

## Installation Instructions
To install this project, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/ms347135/genai-video-startups-data-analysis.git
   ```
2. Navigate into the project directory:
   ```bash
   cd genai-video-startups-data-analysis
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage Examples
Here are some usage examples to help you get started:
- **Analyze Market Trends:**  
   Run the following command to get the latest market analysis:
   ```bash
   python analyze.py --market
   ```
- **Generate Reports:**  
   You can generate reports by executing:
   ```bash
   python report.py --generate
   ```
